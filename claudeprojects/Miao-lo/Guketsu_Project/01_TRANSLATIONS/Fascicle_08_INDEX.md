# Fascicle 8: Index
## Zhiguan Fuxing Zhuan Hongjue (止觀輔行傳弘決) — The "Repair Manual"

**Translator**: William Altig / Claude
**Date**: January 23, 2026

---

## Overview

Fascicle 8 covers the **Internal Obstacles** — Objects 2-4 of the Ten Objects: Afflictions (煩惱境), Illness (病患境), and Karma (業境). It details how these obstacles arise during practice and how to apply both gradual (specific antidotes) and sudden (seeing empty nature) methods to work with them.

## The Pattern

Every obstacle gets the same treatment:

| Approach | Method | Result |
|---|---|---|
| **Gradual** | Apply specific medicine to specific disease | Symptoms addressed |
| **Sudden** | See the empty nature directly | Root cut |
| **Integration** | Use both as appropriate | Complete practice |
