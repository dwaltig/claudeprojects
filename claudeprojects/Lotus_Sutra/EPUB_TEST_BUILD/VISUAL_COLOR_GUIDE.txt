================================================================================
USAGE GUIDE
================================================================================

USE BRIGHT GOLD (#FFD700) FOR:
├─ Main headers (h1, h2, h3)
├─ Sanskrit terms and keywords
├─ Important emphasis
├─ Links and references
├─ Blockquote and section borders
├─ Visual prominence
└─ Maximum accessibility

USE GOLDENROD (#DAA520) FOR:
├─ Secondary elements
├─ Visited link states
├─ Subtitle text
├─ Metadata
├─ Hover backgrounds (with white text)
└─ Complementary accents

USE DARK CHARCOAL (#1a1a1a) FOR:
├─ Table header backgrounds only
├─ Creates strong reversal with bright gold text
└─ Achieves maximum contrast

KEEP CREAM (#fffef5) FOR:
├─ Body background (unchanged)
├─ Warm, inviting reading experience
├─ Pairs perfectly with gold accents
└─ Part of original design philosophy
