## How to Choose Which Version

### **Use PUBLICATION EDITION if:**
- You want to read the pure translation without apparatus
- You're publishing it as a book or ebook
- You want a clean, minimal text
- You're doing dramatic performance
- You just want the Dharma without commentary
- You prefer uninterrupted reading

### **Use ENHANCED EDITION if:**
- You're studying Buddhism or the Lotus Sutra
- You're in a book club or study group
- You want context and guidance
- You're learning about gender and Buddhism
- You're preparing to perform and want notes
- You want to understand Buddhist terminology
- You're exploring spiritual practice
- You want discussion prompts for reflection

### **Use BACKUP EDITION if:**
- You're researching the editorial process
- You want to see how corrections were made
- You're doing academic analysis
- You need version comparison
- You're archiving different states

## Content Differences Explained

### **The Core Translation (28 Chapters)**
- IDENTICAL in both Publication and Enhanced versions
- Same text, same pacing spaces, same corrections
- No changes between them in the actual translation

### **Supporting Materials (Enhanced Only)**
- Introduction: Scholarly context and orientation

### **Reading Experience Difference**
- **Publication:** "Here is the text. Read it."

Neither is better—they serve different purposes.
