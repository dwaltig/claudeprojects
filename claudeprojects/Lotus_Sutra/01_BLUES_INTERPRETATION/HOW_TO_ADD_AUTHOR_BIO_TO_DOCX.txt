================================================================================
HOW TO ADD AUTHOR BIOGRAPHY TO YOUR DOCX
Quick Guide for Integrating Author Bio into the Document
================================================================================

STEP-BY-STEP INSTRUCTIONS
================================================================================

STEP 1: OPEN THE DOCX FILE
───────────────────────────

File to open: lotus_sutra_for_elevenreader_FORMATTED.docx

Open in one of these applications:
- Microsoft Word (Windows or Mac)
- Google Docs (drag file in, or File > Open)
- Pages (Mac - click "Open File")
- LibreOffice Writer (free)

================================================================================

STEP 2: NAVIGATE TO THE END OF THE DOCUMENT
────────────────────────────────────────────

Press: Ctrl+End (Windows) or Command+End (Mac)

This takes you to the very end of the document, after all chapter content.

================================================================================

STEP 3: CREATE A NEW PAGE FOR AUTHOR BIO
────────────────────────────────────────

Position your cursor at the end of the document.

Press: Ctrl+Enter (Windows) or Command+Enter (Mac)

This creates a page break for the new "About the Author" section.

================================================================================

STEP 4: ADD THE SECTION HEADER
──────────────────────────────

Type: About the Author

Format as follows:
1. Highlight the text "About the Author"
2. Make it a Heading 2 (H2) style to match other section headers:
   - In Word: Home tab → Styles → Heading 2
   - In Google Docs: Format → Text → Heading 2
   - In Pages: Click paragraph style dropdown → Heading 2
3. Optional: Make it larger and bold for emphasis
   - Font size: 16-18pt (or use Heading 2 default)
   - Bold: Yes (apply bold formatting)

Optional: Add two line breaks above for spacing

================================================================================

STEP 5: CHOOSE YOUR BIOGRAPHY VERSION & ADD IT
──────────────────────────────────────────────

Open: AUTHOR_BIOGRAPHY_FOR_FRONT_MATTER.txt

Choose one of these versions based on your needs:

RECOMMENDED FOR AUDIOBOOK:
→ Use "SHORT BIO" (150-200 words)
   Professional, concise, perfect for digital formats

RECOMMENDED FOR PRINT BOOK:
→ Use "EXTENDED SHORT BIO" (250-300 words)
   More detailed, suitable for printed book back matter

RECOMMENDED FOR COMPREHENSIVE:
→ Use "FULL-LENGTH BIOGRAPHY" (400-500 words)
   Complete background, ideal for scholarly or detailed contexts

AFTER CHOOSING:

1. Copy your selected biography text (Ctrl+C / Command+C)
2. In your DOCX document, position cursor after "About the Author" heading
3. Press Enter to create a new paragraph
4. Paste the text (Ctrl+V / Command+V)

================================================================================

STEP 6: FORMAT THE BIOGRAPHY TEXT
──────────────────────────────────

The biography text should be:
- Normal paragraph style (not bold, not heading)
- Font: Calibri, 11pt (same as rest of document body text)
- Alignment: Left align (default)
- Single spacing (default)
- One blank line between paragraphs if there are multiple paragraphs

If your document uses different formatting:
1. Select all biography text (Ctrl+A while in that section)
2. Apply "Normal" style from the styles menu
3. Font size: 11pt
4. Font: Calibri or your document's body font

================================================================================

================================================================================

STEP 8: SAVE YOUR DOCUMENT
──────────────────────────

Save the file:
- Word: File → Save (or Ctrl+S)
- Google Docs: Auto-saves, but manually save with Ctrl+S to be sure
- Pages: File → Save
- LibreOffice: File → Save

Keep the same filename: lotus_sutra_for_elevenreader_FORMATTED.docx

================================================================================

STEP 9: VERIFY THE INTEGRATION
──────────────────────────────

Scroll through the document to make sure:
✓ All 28 chapters are still intact
✓ All page breaks are still in place
✓ Author bio section appears at the very end
✓ "About the Author" header is formatted as H2
✓ Biography text is readable and properly formatted
✓ No content was accidentally deleted
✓ Document still opens properly in Word/Google Docs

TROUBLESHOOTING
================================================================================

PROBLEM: Document seems corrupted after adding bio
SOLUTION:
- Undo changes (Ctrl+Z) and try again
- If still having issues, use the QUICK_START_GUIDE.txt to regenerate

PROBLEM: Page breaks moved around
SOLUTION:
- This is normal when adding content
- Verify your bio is on its own page at the end
- If breaks are scattered, you may need to manually move page breaks

PROBLEM: Formatting looks weird (fonts changed, spacing off)
SOLUTION:
- Select biography text
- Clear all formatting: Ctrl+M (Word) or Format > Clear formatting (Google Docs)
- Reapply Normal style
- Font should return to Calibri 11pt

PROBLEM: Cannot open document after editing
SOLUTION:
- Close without saving (don't save the broken version)
- Open original DOCX again
- Try again with more careful editing

PROBLEM: Biography text has different style than document
SOLUTION:
- Make sure to use Normal paragraph style, not Heading or other style
- Match font to document body: typically Calibri or Arial, 11pt
- Remove any bold, italic, or special formatting
