## 3. PARAGRAPH STYLES & FORMATTING

- **Alignment:** CENTER
- **Space Before:** 0pt (first chapter) / 18pt (mid-document)
- **Space After:** 12pt
- **Page Break Before:** YES (except first chapter)
- **Keep with Next:** YES

### Heading 2 (Section Headers)
- **Font:** Garamond 14pt, Bold
- **Line Spacing:** 16.8pt (1.2x)

- **Space Before:** 12pt

- **Keep with Next:** YES

- **Alignment:** Left
- **Space Before:** 6pt

- **Alignment:** CENTER
- **Space Before:** 0pt
- **Space After:** 12pt
- **Keep with Next:** YES

## 4. CHAPTER STRUCTURE TEMPLATE

Every chapter must follow this structure exactly:

```
[Page Break]

[1] CHAPTER [NUMBER]: [TITLE]              [Heading 1, CENTER]
[2] A Blues Translation from Kumārajīva's Classical Chinese (406 CE)    [Byline, CENTER, ITALIC]
[3] [blank line - space_after = 12pt]
[4] [SECTION HEADER or first content]      [Heading 2 if header]
[5+] Chapter content flows...              [Normal + Verse as appropriate]
...
[N-3] ***                                   [Separator - CENTER aligned, Normal]
[N-2] [Final verse line]
[N-1] [blank line - space_after = 18pt]
[N] END OF CHAPTER [#]: [TITLE]            [Normal]
[N+1] [blank line - space_after = 18pt]
[N+2] INTERPRETATION NOTES                 [Heading 2]
[N+3+] Interpretation content...           [Interpretation Notes Body style]
```

**Key Rules:**
- Every chapter starts on new page (Page Break Before on Heading 1)
- No blank lines should be empty paragraphs - use space_after instead
- *** separators mark verse-to-prose transitions (CENTER aligned, Normal style)
- Interpretation notes always at end of chapter
- Consistent spacing throughout

## 5. CHAPTER COMPONENTS

### Chapter Title (Heading 1)
- Format: `CHAPTER [#]: [FULL TITLE IN UPPERCASE]`
- Examples:
  - `CHAPTER ONE: THE OPENING`
  - `CHAPTER SEVEN: THE PHANTOM CITY BLUES`
  - `CHAPTER TWENTY-EIGHT: UNIVERSAL WORTHY COMES TO THE ASSEMBLY`
- Always uppercase
- Always centered
- Always on new page

### Byline
- Format: `A Blues Translation from Kumārajīva's Classical Chinese (406 CE)`
- Identical for all chapters
- Always italic
- Always centered
- Always appears directly after title (para +1)
- Spacing after: 12pt

### Content Sections
- **Prose sections:** Heading 2 for section titles
- **Verse sections:** Use Verse style, indent 0.5" for visual effect
- **Between prose and verse:** Add *** separator (CENTER, Normal style)
- **No blank lines:** All spacing via space_after property

### Verse/Prose Separators
- Format: `***` on own paragraph
- Style: Normal (11pt Garamond)
- Alignment: CENTER
- Use: Between verse and prose transitions only
- Space before: 6pt
- Space after: 6pt

### END OF CHAPTER Marker
- Format: `END OF CHAPTER [#]: [TITLE]`
- Examples:
  - `END OF CHAPTER 1: THE OPENING`
  - `END OF CHAPTER 7: THE PHANTOM CITY BLUES`
- Style: Normal (11pt Garamond)
- Alignment: Left
- Spacing after: 18pt

### INTERPRETATION NOTES Section
- Header: `INTERPRETATION NOTES` (Heading 2 style)
- Always present at end of chapter
- Content: Multiple paragraphs of analysis
- Style: Interpretation Notes Body
- Font: 10.5pt Garamond (slightly compressed)
- Typical length: 200-400 words per chapter

## 6. TEXT FORMATTING WITHIN CONTENT

### Sanskrit Diacriticals (CRITICAL)
- **Always preserve:** All Unicode characters must remain unchanged
- **Key characters:** ā, ī, ū, ṛ, ṇ, ñ, ś, ṣ, ḍ, ṭ, ṃ, ṅ
- **Names to preserve exactly:**
  - Śāriputra (not Sariputra)
  - Mañjuśrī, Mahākāśyapa, Avalokiteśvara
  - Ājñāta-Kauṇḍinya
- **Font handling:** Garamond supports all Sanskrit diacriticals
- **Print test:** Before final export, verify all diacriticals render correctly

### Bold Text
- **Use for:** Buddha names on first occurrence per chapter
- **Examples:** **Śāriputra**, **Tathāgata**, **Mañjuśrī**
- **Method:** Use native Word bold (Ctrl+B), not markdown `**`

### Italic Text
- **Use for:** Technical terms requiring emphasis
- **Examples:** *dharma*, *nirvāṇa*, *samādhi*, *bodhisattva*
- **Use for:** Chapter bylines (all italic)
- **Method:** Use native Word italic (Ctrl+I), not markdown

### NO Markdown Formatting
- **PROHIBITED:** `**bold**` markdown syntax
- **PROHIBITED:** `*italic*` markdown syntax
- **PROHIBITED:** `##` heading markers
- **PROHIBITED:** `***` as markdown (use as separator paragraph only)
- **All formatting via:** Native Word styles and properties

## 7. SPECIAL ELEMENTS

### Opening Verses
- First verse in many chapters starts with traditional formula
- Use Verse style (1.5x line spacing)
- Indent: 0.5"

### Dialogue/Speeches
- Buddhist dialogues: Use Normal style (body text)
- Long speeches: Can use indentation or Verse style
- Quotations: Use quotation marks, not block quotes

### Lists & Enumerations
- **Bullet points:** In interpretation notes only
- **Numbered lists:** In interpretation notes only
- **Style:** Use native Word bullets/numbering

### Horizontal Rules/Separators
- **Method:** Use `***` paragraph (CENTER, Normal style)
- **Placement:** Between verse and prose sections
- **Styling:** No additional formatting needed

## 13. KDP/INGRAMPSPARK REQUIREMENTS

### File Format
- **Accepted:** DOCX (Microsoft Word) or PDF
- **Recommended:** PDF (more reliable formatting preservation)
- **Export setting:** High quality

### ISBN & Metadata
- **ISBN:** Obtain from KDP/IngramSpark
- **BISAC Category:** Religion / Buddhism
- **Keywords:** Lotus Sutra, Buddhist Scripture, Blues Translation, Poetry
- **Description:** Include in KDP dashboard (not in document)

### Cover Design
- **Dimensions:** 6" x 9" (trim size) + bleed
- **File format:** PDF or high-resolution image
- **Separate file:** Design outside of manuscript

### Print Quality
- **Proof review:** Always order proof copy before full print
- **Font rendering:** Verify Garamond renders clearly
- **Page breaks:** Confirm chapters start on correct pages
- **Margins:** Verify text doesn't print into margins

## 14. EXPORT SPECIFICATIONS

### PDF Export Settings
- **Resolution:** 300 DPI (for print quality)
- **Color:** Grayscale / Pure Black
- **Embed fonts:** YES (all fonts)
- **Compress:** Lossless
- **Page size:** 6" x 9"

### EPUB Export (Future)
- **Flow:** Reflowable
- **CSS:** Semantic styles (body, h1, h2, verse, etc.)
- **Images:** Optional
- **Include cover:** Yes

### Print PDF Steps
1. Save document as DOCX
2. File → Export as PDF
3. Choose "Standard" or "High Quality"
4. Verify settings match above
5. Review PDF in Adobe Reader
6. Upload to KDP/IngramSpark
