## Methodology

### Source Reference Database

The following Buddhist textual sources were used as citation references:

1. **Tathagatagarbha-sutra**: Buddha-nature, inherent potential, awakening
2. **Lotus Sutra itself**: Skillful means, prophecy, dharma-teacher status, one-vehicle doctrine
3. **Abhidharma-Kosa**: Arhat analysis, cosmology, path structures, technical terms
4. **Bodhisattva-bhumi**: Bodhisattva practice, six perfections, merit-generation
5. **Pali Canon / Samyutta Nikaya**: Four Noble Truths, Dependent Origination
6. **Mahaprajnaparamita Sutra**: Emptiness, wisdom, synthesis
7. **Pure Land Sutras**: Buddha-realm descriptions, environmental manifestation
8. **Yogacara texts**: Consciousness, mind-only doctrine
9. **Madhyamaka texts**: Two truths, emptiness doctrine
10. **Avatamsaka/Huayan**: Cosmology, interconnection, systems
11. **Mahayana universal**: Merit transference, universal salvation, dharani

### Citation Format

Citations were added using the following format patterns:

- **"According to [Source],"** - Direct doctrinal correspondence
- **"Following [Source]'s understanding of [concept],"** - Conceptual framework alignment
- **"Drawing from [Tradition]'s framework of [concept],"** - Broader tradition reference

### Placement

Citations were inserted at the beginning of footnote explanations, immediately after any bold term definition, preserving all original content.

### Quality Standards

1. **No duplicate citations**: Each footnote checked for existing citations before addition
2. **Content preservation**: No alteration of meaning or existing text
3. **Logical matching**: Citations matched to doctrinal content of footnote
4. **Parallel structure**: Similar footnotes received parallel citation patterns

## Methodology Documentation

### For Future Contributors

To continue this work on remaining chapters (3-28):

1. **Open target chapter file**: `CHAPTER_NN_[TITLE].md`

2. **Locate Apparatus Summary section**: Search for `## Apparatus Summary`

3. **Identify all footnote entries**: Look for numbered bold entries (e.g., `1. **Term**:`)

4. **For each footnote**:
   - Check if citation already exists (search for "According to", "Following", "Drawing from")
   - If no citation present, analyze doctrinal content
   - Match to appropriate source from database
   - Insert citation at beginning of explanation after bold term
   - Preserve all original content

5. **Source matching logic**:
   - **Cosmology** → Abhidharma-Kosa
   - **Buddha-nature** → Tathagatagarbha-sutra
   - **Wisdom/Emptiness** → Mahaprajnaparamita Sutra or Madhyamaka
   - **Consciousness** → Yogacara
   - **Dependent Origination/Four Noble Truths** → Pali Canon / Samyutta Nikaya
   - **Bodhisattva path/Six Perfections** → Bodhisattva-bhumi
   - **Skillful means/One vehicle** → Lotus Sutra itself
   - **General Mahayana** → Mahayana universal tradition
   - **Prophecy/Merit transference/Dharani** → Mahayana universal tradition

6. **Quality check**: After completing each chapter, verify:
   - All footnotes have citations
   - No duplicate citations within same footnote
   - Format consistency maintained
   - Original content unchanged
