# Vimalakirti Sutra Translation Project

A dual-translation project for the Vimalakirti Sutra (維摩詰所說經), producing both scholarly and Blues vernacular versions.

2. **Check** `03_DOCUMENTATION/COMPLETION_STATUS.md` for current progress
3. **Review** agent profiles in `04_AGENTS/`

## The Vimalakirti Sutra

**Source**: Kumarajiva translation (T0475)  
**Structure**: 14 chapters, 3 volumes  
**Theme**: Non-duality, lay Buddhism, skillful means  
**Famous Scene**: Vimalakirti's "thunderous silence" (Chapter 9)

## Translation Approach

### Dual Translation
- **Scholarly**: Academic English with extensive footnotes
- **Blues Vernacular**: African-American Blues idiom

### Dual Agent System
- **The Professor**: Creates scholarly translation
- **The Bluesman**: Reviews and creates Blues version
