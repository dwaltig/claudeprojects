## üß† Reasoning Frameworks Reference
When invoking these agents, you are engaging specific mental models:

1.  **Reflexion (The Professor):** Best for high-accuracy translation. The agent drafts, critiques its own work against source material, and produces a final version.
2.  **ReAct (The Bluesman):** Best for creative interpretation. The agent thinks about the "soul" of the text, acts by translating it into Blues idiom, and observes the result.
3.  **A-Mem (Digital Archivist):** Uses a Zettelkasten-style atomic memory system to ensure long-term consistency across different sutra projects.
4.  **Epistemic Auditing (Rationalist Reviewer):** Analyzes text for hidden assumptions, logical fallacies, and certainty-levels.
