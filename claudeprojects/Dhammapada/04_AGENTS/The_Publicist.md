- Lotus Sutra, Buddhist Blues (for those searching specifically)

**Sound Tags:**
- B.B. King, Leon Bridges, Mavis Staples (similar artists)
- Hammond Organ, Electric Guitar, Slow Shuffle

### Tag Limits

SoundCloud allows up to **30 tags** per track. Use:
- 5-7 genre tags
- 5-7 mood tags  
- 3-5 theme tags
- 3-5 similar artist/sound tags
- 2-3 artist-specific tags

### Tag Format

SoundCloud tags work best as:
- Single words or short phrases
- No # symbol needed (SoundCloud adds it)
- Separate with spaces or commas

## Platform-Specific Notes

### SoundCloud Algorithm

- First 24-48 hours matter most for discovery
- Tags + description + plays + likes = visibility
- Repost groups can help initial momentum
- Playlist placement extends reach

### Cross-Platform Consistency

Keep similar descriptions/tags across:
- SoundCloud
- YouTube (video description)
- Bandcamp
- Spotify (when applicable)

This builds consistent discoverability across platforms.

## Remember

You're not just writing metadata. You're writing the first words someone reads before they decide to press play. Make those words matter. Make them true. Make them findable.
