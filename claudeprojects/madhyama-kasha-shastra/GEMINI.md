# GEMINI.md - Madhyamakashāstra Translation Project

## 1. Project Overview

The **Madhyamakashāstra Translation Project** translates Nāgārjuna's *Mūlamadhyamakakārikā* with Candrakīrti's *Prasannapadā* commentary into scholarly English and Blues vernacular.

**Key Components:**
- **Source Text**: Sanskrit (Mithila Institute 1960 edition)
- **Dual Translation**: Scholarly + Blues Vernacular
- **Agent System**: Professor (Dr. Rajesh Patel) + Bluesman (Master John Kim)

## 2. Critical Safety Protocol

- **Preserve Sanskrit**: Handle diacritics carefully (ś, ṣ, ṇ, ā, ī, ū, ṃ, ḥ)
- **Verify Encodings**: UTF-8 mandatory
- **OCR Quality**: Source text has scan artifacts—verify readings

## 3. Directory Structure

- `CLAUDE.md`: **The Authoritative Guide** - Read this first
- `01_TRANSLATIONS/`: Scholarly, Blues, and Combined versions
- `02_SOURCE_MATERIALS/`: Extracted Sanskrit chapters
- `03_DOCUMENTATION/`: Session notes, glossary, status
- `04_AGENTS/`: Professor and Bluesman system prompts

## 4. Specialized Agents

### The Professor (Dr. Rajesh Patel)
- Sanskrit → Scholarly English with dialectical apparatus
- Identifies opponent schools (Sāṃkhya, Nyāya, etc.)
- Cross-references with Vigrahavyāvartanī, Ratnāvalī

### The Bluesman (Master John Kim)
- Reviews Professor's work for completeness
- Creates Blues vernacular version
- Transforms arguments into liberation narratives

## 5. Key Sanskrit Terms

- **svabhāva** (स्वभाव): Intrinsic nature (what is negated)
- **śūnyatā** (शून्यता): Emptiness
- **pratītyasamutpāda** (प्रतीत्यसमुत्पाद): Dependent origination
- **pratyaya** (प्रत्यय): Condition
- **catuṣkoṭi** (चतुष्कोटि): Tetralemma
