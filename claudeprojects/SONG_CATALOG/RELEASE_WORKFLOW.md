## üöÄ RELEASE WORKFLOW

### Step 1: SoundCloud (Same Day)
1. Upload to SoundCloud
2. Add title, description, and tags
3. Set cover art
4. Add to relevant playlist
5. Copy SoundCloud link

### Step 2: Distribution Submission (Same Day or Next)
Choose ONE distributor:
- **eMastered** ‚Äî Fast, simple singles
- **LANDR** ‚Äî Good for albums, includes mastering
- **TuneCore** ‚Äî Higher upfront cost, 100% royalties

Submit with:
- High-quality audio file
- Cover art (3000x3000px minimum)
- Release date (recommend 2-3 weeks out)
- ISRC code (distributor provides)

### Step 3: Pre-Release Content (While Waiting)
- [ ] Create YouTube visualizer or lyric video
- [ ] Schedule Instagram teaser post
- [ ] Write press release / blog post
- [ ] Update catalog spreadsheet with "Pending" status

### Step 4: Release Day
- [ ] Confirm song is live on all platforms
- [ ] Post to all social channels
- [ ] Update catalog with streaming links
- [ ] Share to relevant playlists
- [ ] Engage with comments
