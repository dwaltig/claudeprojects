# GEMINI.md - Bodhicaryāvatāra Translation Project

## 1. Project Overview

The **Bodhicaryāvatāra Translation Project** translates Śāntideva's *A Guide to the Bodhisattva's Way of Life* (8th century CE Sanskrit) into scholarly English and Blues vernacular.

**Key Components:**
- **Source Text**: Sanskrit (IAST, La Vallée Poussin critical edition)
- **Dual Translation**: Scholarly + Blues Vernacular
- **Agent System**: Professor (Dr. Rajesh Patel) + Bluesman (Master John Kim)

## 2. Critical Safety Protocol

- **Preserve Scholarship**: Handle all text with extreme care
- **Verify Encodings**: UTF-8 with Sanskrit diacritics (ś, ṣ, ñ, ā, ī, ū, ṃ, ḥ)
- **Ask First**: Confirm structural changes with user

## 3. Directory Structure

- `CLAUDE.md`: **The Authoritative Guide** - Read this first
- `01_TRANSLATIONS/`: Scholarly, Blues, and Combined versions
- `02_SOURCE_MATERIALS/`: Sanskrit source (10 chapters)
- `03_DOCUMENTATION/`: Session notes, glossary, status
- `04_AGENTS/`: Professor and Bluesman system prompts
- `05_QA/`: DeepSeek audit logs

## 4. Specialized Agents

### The Professor (Dr. Rajesh Patel)
- Sanskrit → Scholarly English with philological apparatus
- Metrical analysis (anuṣṭubh, triṣṭubh)
- Cross-reference with Crosby/Skilton, Wallace translations

### The Bluesman (Master John Kim)
- Reviews Professor's work for missed nuances
- Creates Blues vernacular version
- Thematic verse groupings

## 5. Translation Workflow

1. Professor translates Sanskrit → Scholarly English
2. Bluesman reviews for completeness
3. Bluesman creates Blues version
4. QA audit via DeepSeek/Rationalist Reviewer
5. Consolidate into Combined files

## 6. Key Sanskrit Terms

- **bodhicitta** (बोधिचित्त): Awakening mind
- **kṣānti** (क्षान्ति): Patience, forbearance
- **vīrya** (वीर्य): Heroic effort, diligence
- **prajñā** (प्रज्ञा): Wisdom, insight
- **pariṇāmanā** (परिणामना): Dedication of merit
