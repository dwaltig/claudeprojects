# CLAUDE.md - Bodhicaryāvatāra Translation Project

## Project Overview

The **Bodhicaryāvatāra Translation Project** is a scholarly and Blues vernacular dual-translation of Śāntideva's 8th-century masterwork, *A Guide to the Bodhisattva's Way of Life*.

**Key Components:**
- **Source Text**: Sanskrit (IAST transliteration, La Vallée Poussin edition)
- **Dual Translation**: Scholarly (Academic) + Blues Vernacular (Accessible)
- **Structure**: 10 chapters, ~912 verses total
- **Agent System**: Professor (Dr. Rajesh Patel) + Bluesman (Master John Kim)

---

## Critical Safety Protocol

**READ BEFORE EDITING:**
- **Preserve Scholarship**: Handle all text with extreme care
- **Verify Encodings**: Ensure UTF-8 and correct Sanskrit diacritics (ś, ṣ, ñ, ā, ī, ū, ṃ, ḥ)
- **Ask First**: Confirm structural changes with the user
- **Document Everything**: Update session notes and status files

## Translation Workflow

1. **Professor**: Sanskrit → Scholarly English (thematic verse groupings)
2. **Bluesman**: Reviews for missed nuances, creates Blues version
3. **QA**: Rationalist Reviewer / DeepSeek audit
4. **Consolidation**: Combined markdown file per chapter

## Special Considerations

### Sanskrit Specifics
- **Metrical Poetry**: All content is verse (anuṣṭubh, triṣṭubh meters)
- **IAST Diacritics**: Preserve ś, ṣ, ñ, ā, ī, ū, ṃ, ḥ precisely
- **Compound Words**: Sanskrit compounds require careful unpacking

### Blues Adaptation
- **Confessional Tone**: Chapter 2 aligns with Blues "burden" tradition
- **Devotional Energy**: More personal than dialogic sutras
- **Thematic Grouping**: Verses grouped by theme, not line-by-line
