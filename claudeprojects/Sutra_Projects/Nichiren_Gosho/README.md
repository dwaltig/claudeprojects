# Nichiren Gosho Project

## Overview

This directory is for collecting and studying Nichiren Shōnin's writings (御書, Gosho) from the 13th century. Nichiren (1222–1282) wrote extensively on the Lotus Sutra, making this collection directly relevant to the Lotus Sutra translation project.

## Directory Structure

```
Nichiren_Gosho/
├── 01_ORIGINAL_JAPANESE/    # Original Japanese texts (古語体)
├── 02_TRANSLATIONS/         # English and modern Japanese translations
├── 03_DOCUMENTATION/        # Study materials, glossaries, notes
└── README.md
```

## Primary Sources for Original Japanese Gosho

### 1. Soka Gakkai Gosho Zenshu (創価学会版 御書全集)

**Full-text searchable database:**
- **URL**: https://sokanet.jp/kaiin/study/gosho/gosho_search/
- Contains the complete *Nichiren Daishonin Gosho Zenshu New Edition* (日蓮大聖人御書全集 新版)
- Text in classical Japanese (古語体) as Nichiren wrote it
- Requires Soka Gakkai membership for full access

### 2. PDF Downloads with Furigana

**Complete Gosho Zenshu in PDF format:**
- **URL**: https://js-hpbs.jp/gosho/
- Full collection with furigana (reading guides)
- Free access

### 3. Nichiren Library (English Translations)

**Searchable English translations:**
- **URL**: https://www.nichirenlibrary.org/
- *Writings of Nichiren Daishonin* Vols. I & II
- Cross-referenced with original Japanese titles

### 4. Academic Critical Edition

**Risshō University Edition:**
- *昭和定本日蓮聖人遺文* (Shōwa Teibon Nichiren Shōnin Ibun)
- Critical apparatus with variant readings
- Standard academic reference

## Connection to Lotus Sutra Project

Nichiren's writings are essentially commentaries on the Lotus Sutra. Key connections:

1. **Lotus Sutra Chapters Discussed**: Nichiren frequently cites Chapters 2 (Skillful Means), 10 (Dharma Teachers), 11 (Treasure Tower), 16 (Lifespan of the Tathāgata), 21 (Supernatural Powers), 25 (Universal Gate)

2. **Tiantai/Tendai Lineage**: Nichiren follows Zhiyi's interpretation of the 28-chapter structure (Trace/Fundamental)

3. **Daimoku Practice**: The chanting of *Namu Myōhō Renge Kyō* as the essential practice of the Lotus Sutra

## Download Instructions

3. Copy relevant passages for study
