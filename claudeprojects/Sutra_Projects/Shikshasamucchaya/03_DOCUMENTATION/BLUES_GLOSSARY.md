## Usage Notes

### Voice Guidelines
1. **Direct Address**: Use "you" consistently, not "one should"
2. **Call & Response**: "You hear that?", "You understand?", "Listen:"
3. **Concrete Logic**: Use physical metaphors, avoid abstract jargon
4. **Rhythm**: Short sentences, repetition, caesura for emphasis
5. **Blues Echoes**: Biblical cadence, Delta references when appropriate

### Metaphor Selection
- Choose metaphors from **Blues Universe**: Delta, Storefront Church, Jazz Club, Chain Gang, The Road
- Root metaphor should be chosen **before** writing each chapter
- Metaphors should be **sustained** throughout the chapter, not one-off

### Term Consistency
- Check this glossary before translating each new section
- Update glossary immediately after completing a chapter
- Flag any new creative translations for team review
