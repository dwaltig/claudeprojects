# GEMINI.md - Shikshasamucchaya Project

## 1. Project Overview
The **Shikshasamucchaya Project** focuses on Śāntideva's *Śikṣāsamuccaya*, a compendium of Mahāyāna thought and practice. It is the sister text to the *Bodhicaryāvatāra*.

## 2. Directory Structure
- `01_TRANSLATIONS/`: Translations (Scholarly/Blues)
- `02_SOURCE_MATERIALS/`: Source texts (Sanskrit)
- `03_DOCUMENTATION/`: Notes and tracking
- `04_AGENTS/`: Agent definitions

## 3. Specialized Agents
*   **The Professor**: Scholarly translation, philological focus.
*   **The Bluesman**: Vernacular/Blues interpretation.

## 4. Critical Safety Protocol
- **Preserve Scholarship**: Handle Sanskrit with care.
- **Verify Encodings**: UTF-8 is mandatory.
- **Diacritics**: Ensure correct IAST (ā, ī, ū, ṛ, ṝ, ḷ, ḹ, ṅ, ñ, ṇ, ṭ, ḍ, ś, ṣ, ṃ, ḥ).

- **Language**: Sanskrit (Source) -> English (Target)
